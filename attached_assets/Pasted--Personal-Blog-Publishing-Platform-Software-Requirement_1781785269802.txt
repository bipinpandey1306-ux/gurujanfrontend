## Software Requirement Document (SRD)

### Project Name

Personal Blog CMS

### Project Owner

Blog Author (Father)

### Version

1.0

---

# Project Overview

एक आधुनिक Personal Blogging Platform विकसित की जानी है जहाँ वेबसाइट के Owner स्वयं अपने विचार, लेख, अनुभव, कहानियाँ, धार्मिक लेख, सामाजिक लेख, समाचार विश्लेषण तथा अन्य सामग्री प्रकाशित कर सकें।

सिस्टम इतना सरल होना चाहिए कि बिना किसी Technical Knowledge के Admin आसानी से Content Manage कर सके।
